# RCS - Raspberry Control Script

## Goal
- Improve my Python skills.
- Create a simple raspberry management script.

## How To Use
- Dowload or clone the repo 
- Assign the execute permission to RCS.py with sudo chmod +x RCS.py 
- Run RCS with ./RCS.py 
- Enjoy the script
